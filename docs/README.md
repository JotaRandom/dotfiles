Documentación del repositorio

Este repositorio usa una estructura híbrida con módulos reaprovechables y un área para archivos específicos de máquinas. Consulta `STRUCTURE.md` para ver la arquitectura y `CONTRIBUTING.md` para normas de colaboración.

Consejos rápidos:
- Usa `git status` y `git diff` antes de mover archivos.
- Abre un PR para cambios globales en la estructura para recibir feedback.
