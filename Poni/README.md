Poni — Imágenes para ponysay

Este directorio contiene una colección de imágenes (formatos PNG/GIF/XCF y otros) utilizadas con `ponysay` u otros scripts de personalización.

Importante (Licencias y atribuciones)
- Las imágenes incluidas provienen de autores diversos: algunas son obra original del mantenedor, otras son fan art o imágenes tomadas de plataformas como DeviantArt o colecciones públicas.
- Antes de redistribuir cualquier imagen, verifica si el autor incluye una licencia o una nota de atribución en su archivo, o consulta la fuente original cuando esté disponible en el comentario del archivo (por ejemplo, nombres de archivos con referencias a autores).
- Si deseas usar, remezclar o publicar estas imágenes, respeta las licencias de los autores y agrégales la atribución correspondiente. En caso de duda, contacta al autor para pedir permiso.

Cómo contribuir
- Si tienes permisos de un autor y deseas añadir imágenes aquí, añade un archivo de texto con la atribución y/o licencia en el mismo directorio con el mismo nombre (por ejemplo `miimagen.png` y `miimagen.license.txt`).
- Si encuentras una imagen que requiere correcciones o eliminación por cuestiones de copyright, abre un issue o un pull request explicando el problema.

Nota histórica
- Originalmente estas imágenes estaban en `Toshiba/poni`; se movieron a `Poni/` para centralizarlas (2025-11-27).
