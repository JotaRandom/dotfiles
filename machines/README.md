Carpeta `machines/`

Contiene archivos específicos de cada máquina:
- `machines/L420/` — configuración y archivos específicos del Lenovo ThinkPad L420 (hardware dumps, DSDT, etc.).
- `machines/Toshiba/` — configuraciones históricas de la antigua Toshiba.

Nota: No se recomienda stowear directamente estas carpetas; usa `modules/` para configuraciones que quieras desplegar en una máquina.
